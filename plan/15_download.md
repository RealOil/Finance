## 작업 내용

### 1. 다운로드 데이터 생성 모듈

**파일**: `modules/download.py`

**기능**:
- 입력 데이터와 계산 결과를 결합하여 다운로드용 데이터 생성
- JSON 형식 데이터 생성
- CSV 형식 데이터 생성 (시나리오 비교 등)

### 2. 다운로드 UI 컴포넌트

**위치**: 각 결과 페이지 (페이지 1, 2, 3)

**기능**:
- JSON 다운로드 버튼
- CSV 다운로드 버튼 (해당되는 경우)
- 다운로드 파일명에 타임스탬프 포함

### 3. 다운로드 데이터 구조

**JSON 구조**:
```json
{
  "입력 데이터": {...},
  "계산 결과": {...},
  "계산 일시": "...",
  "데이터 출처": "..."
}
```

**CSV 구조**:
- 시나리오 비교 결과를 테이블 형식으로
