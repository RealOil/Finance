## 작업 내용

### 1. 페이지 구성

**파일**: `app.py`

**섹션 구성**:
1. **헤더**
   - 프로젝트 제목
   - 간단한 설명

2. **프로젝트 소개**
   - 목적 및 배경
   - 주요 기능 소개

3. **빠른 시작 가이드**
   - 사용 방법 안내
   - 단계별 가이드

4. **기능 미리보기**
   - 3개 주요 페이지 소개
   - 각 페이지의 기능 요약

5. **면책 조항**
   - 데이터 출처 안내
   - 면책 조항 표시

### 2. UI 구성 요소

- 제목 및 헤더 (st.title, st.header)
- 설명 텍스트 (st.markdown)
- 정보 박스 (st.info)
- 경고 박스 (st.warning)
- 구분선 (st.divider 또는 st.markdown("---"))
- 사이드바에 입력 폼 표시 (shared/input_form 사용)

### 3. 세션 상태 초기화

- `shared.session_manager.init_session_state()` 호출
- 입력 폼 렌더링
