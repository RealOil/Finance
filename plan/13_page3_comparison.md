## 작업 내용

### 1. 페이지 구성

**파일**: `pages/3_시나리오_비교.py`

**섹션 구성**:
1. **페이지 헤더**
   - 제목 및 설명

2. **시나리오 입력**
   - 기본 시나리오 (현재 패턴 유지)
   - 비교할 시나리오 입력 (예: "지출 -10%", "연봉 +5%")

3. **시나리오 비교 결과**
   - 각 시나리오별 미래 자산
   - 최적/최악 시나리오 표시
   - 차이점 분석

4. **시각화**
   - 시나리오별 자산 변화 비교 차트

5. **인사이트 텍스트**
   - 시나리오 비교 인사이트
   - 권장 사항

6. **데이터 출처 및 면책 조항**
   - 사용된 데이터 출처 표시
   - 면책 조항

### 2. 주요 기능

- `shared.input_form.render_input_form()` 호출하여 입력 폼 표시
- `shared.session_manager.get_shared_inputs()`로 입력 데이터 가져오기
- `modules.validators.validate_inputs()`로 입력 검증
- 시나리오 입력 UI (텍스트 입력 또는 선택)
- `modules.calculations.parse_scenario()`로 시나리오 파싱
- `modules.calculations.calculate_scenario()`로 각 시나리오 계산
- `modules.calculations.compare_scenarios()`로 시나리오 비교
- `modules.visualizations.create_scenario_comparison_chart()`로 비교 차트 생성
- 최적/최악 시나리오 표시

### 3. 시나리오 입력 형식

- "지출 -10%": 월 지출 10% 감소
- "연봉 +5%": 연봉 증가율 5% 증가
- "자산 +1000": 현재 자산에 1000만원 추가
- "부채 -500": 현재 부채에서 500만원 감소

### 4. 에러 처리

- 입력 검증 실패 시 경고 메시지 표시
- 시나리오 파싱 오류 시 안내 메시지 표시
- 계산 오류 시 안내 메시지 표시
