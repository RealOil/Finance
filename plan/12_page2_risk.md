## 작업 내용

### 1. 페이지 구성

**파일**: `pages/2_리스크_시나리오.py`

**섹션 구성**:
1. **페이지 헤더**
   - 제목 및 설명

2. **입력 데이터 요약**
   - 현재 입력된 데이터를 요약 표시

3. **리스크 분석 결과**
   - 소득 중단 생존 기간
   - 경제 위기 시나리오
   - 은퇴 후 생활 유지 가능 여부
   - 종합 위험도 점수

4. **시각화**
   - 소득 중단 생존 기간 차트
   - 위험도 점수 게이지
   - 위험도 점수 세부 항목 막대 그래프

5. **인사이트 텍스트**
   - 리스크 인사이트
   - 권장 사항

6. **데이터 출처 및 면책 조항**
   - 사용된 데이터 출처 표시
   - 면책 조항

### 2. 주요 기능

- `shared.input_form.render_input_form()` 호출하여 입력 폼 표시
- `shared.session_manager.get_shared_inputs()`로 입력 데이터 가져오기
- `modules.validators.validate_inputs()`로 입력 검증
- `modules.calculations.calculate_income_interruption_survival()`로 소득 중단 생존 기간 계산
- `modules.calculations.calculate_crisis_scenario()`로 경제 위기 시나리오 계산
- `modules.calculations.calculate_retirement_sustainability()`로 은퇴 후 생활 유지 가능 여부 계산
- `modules.calculations.calculate_risk_score()`로 종합 위험도 점수 계산
- `modules.visualizations.create_survival_chart()`로 생존 기간 차트 생성
- `modules.visualizations.create_risk_score_chart()`로 위험도 점수 게이지 생성
- `modules.visualizations.create_risk_breakdown_chart()`로 위험도 점수 세부 항목 차트 생성
- `modules.formatters.generate_risk_insight()`로 리스크 인사이트 생성

### 3. 에러 처리

- 입력 검증 실패 시 경고 메시지 표시
- 계산 오류 시 안내 메시지 표시
