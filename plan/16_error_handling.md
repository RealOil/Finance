## 작업 내용

### 1. 안전한 계산 래퍼 함수

**파일**: `modules/utils.py` (또는 `modules/calculations.py`에 추가)

**기능**:

- `safe_calculate()` 함수: 계산 함수를 안전하게 실행하고 에러 처리
- 예외 발생 시 사용자 친화적인 메시지 반환
- 에러 로깅 (선택적)

### 2. 에러 처리 전략

**에러 유형별 처리**:

1. **입력 검증 오류**: 이미 `validate_inputs()`에서 처리
2. **계산 오류**: ZeroDivisionError, ValueError 등
3. **타입 오류**: 잘못된 데이터 타입
4. **예상치 못한 오류**: 일반 Exception

### 3. 사용자 안내 메시지

**메시지 유형**:

- 에러 메시지: 명확한 오류 설명
- 경고 메시지: 주의가 필요한 상황
- 정보 메시지: 도움말 및 안내

### 4. 각 페이지에 에러 처리 적용

- 페이지 1, 2, 3에 `safe_calculate()` 적용
- 에러 발생 시 계산 중단 및 안내 메시지 표시
