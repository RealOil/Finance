## 작업 목표

모든 페이지에서 공통으로 사용할 수 있는 입력 폼 컴포넌트를 구현합니다.
사이드바에 입력 폼을 렌더링하고, 입력된 데이터를 세션 상태에 저장합니다.

---

## 작업 내용

### 1. 입력 폼 렌더링 함수

**함수**: `render_input_form()`

**기능**:
- 사이드바에 입력 폼 렌더링
- 입력 필드:
  - 기본 정보: 현재 나이, 은퇴 나이
  - 소득 정보: 연봉, 연봉 증가율, 보너스
  - 소비 정보: 월 지출, 연간 고정 지출
  - 자산 및 부채: 총 자산, 총 부채
- 세션 상태에 입력 데이터 저장
- 입력 필드 기본값 설정 (세션 상태에서 가져오기)
- 도움말 텍스트 추가

**반환값**:
```python
{
    'current_age': int,
    'retirement_age': int,
    'salary': float,
    'salary_growth_rate': float,
    'bonus': float,
    'monthly_expense': float,
    'annual_fixed_expense': float,
    'total_assets': float,
    'total_debt': float
}
```

### 2. 입력 필드 구성

**섹션 1: 기본 정보**
- 현재 나이 (number_input, 0-150)
- 은퇴 나이 (number_input, current_age+1 ~ 100)

**섹션 2: 소득 정보**
- 연봉 (number_input, 만원 단위)
- 연봉 증가율 (slider, 0-20%, 기본 3%)
- 보너스 (number_input, 선택, 만원 단위)

**섹션 3: 소비 정보**
- 월 지출 (number_input, 만원 단위)
- 연간 고정 지출 (number_input, 선택, 만원 단위)

**섹션 4: 자산 및 부채**
- 총 자산 (number_input, 만원 단위)
- 총 부채 (number_input, 만원 단위)
