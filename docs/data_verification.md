# 데이터 출처 검증 결과

## 검증 일자

2025년 12월 21일

## 검증 방법

웹 검색을 통한 출처 확인 (구체적 수치 확인 제한적)

---

## 검증 결과 요약

### ✅ 검증 가능한 출처 (기관 존재 확인)

1. **한국은행 경제통계시스템 (ECOS)**

   - ✅ 공식 기관 존재 확인
   - ⚠️ 구체적 수치(2021-2023 평균 2.5%)는 직접 확인 필요
   - **조치 필요**: ECOS 사이트에서 실제 인플레이션 데이터 확인

2. **통계청 국가통계포털 (KOSIS)**

   - ✅ 공식 기관 존재 확인
   - ⚠️ 구체적 수치(임금 상승률, 생명표)는 직접 확인 필요
   - **조치 필요**: KOSIS에서 실제 통계 데이터 확인

3. **금융감독원**

   - ✅ 공식 기관 존재 확인
   - ⚠️ "가계금융조사" 보고서 존재 여부 확인 필요
   - ⚠️ 저축률 30% 기준이 공식 기준인지 확인 필요
   - **조치 필요**: 금융감독원 공식 보고서 확인

4. **국민연금공단**
   - ✅ 공식 기관 존재 확인
   - ⚠️ 기대여명 20년 수치 확인 필요
   - **조치 필요**: 국민연금공단 통계 확인

## ⚠️ 검증이 필요한 항목

### 1. 인플레이션 기본값 2.5%

**현재 문서 내용**:

- "기본값: 2.5% (최근 3년 평균)"
- "참고 기간: 2021-2023년 평균"

**검증 필요 사항**:

- [ ] 한국은행 ECOS에서 2021, 2022, 2023년 소비자물가상승률 확인
- [ ] 3년 평균이 실제로 2.5%인지 계산
- [ ] 2024년 데이터 포함 시 평균이 달라지는지 확인

**권장 조치**:

```
한국은행 ECOS (https://ecos.bok.or.kr/) 접속
→ 통계 → 물가 → 소비자물가상승률
→ 2021, 2022, 2023년 연평균 확인
→ 평균 계산 후 문서 업데이트
```

---

### 2. 재정 건전성 등급 기준 (저축률 30%, 20% 등)

**현재 문서 내용**:

- "금융감독원 가계금융조사: 건강한 가계 기준"
- A등급: 저축률 > 30%
- B등급: 저축률 20-30%

**검증 필요 사항**:

- [ ] 금융감독원에서 "가계금융조사" 보고서 존재 여부 확인
- [ ] 해당 보고서에 저축률 30%, 20% 기준이 명시되어 있는지 확인
- [ ] "건강한 가계 기준"이라는 용어가 공식적으로 사용되는지 확인

**주의사항**:

- 금융감독원 가계금융조사는 실제로 존재하는 보고서인지 확인 필요
- 저축률 기준이 금융감독원 공식 기준인지, 아니면 일반적인 금융 조언인지 구분 필요

**권장 조치**:

```
금융감독원 공식 사이트 (https://www.fss.or.kr/) 접속
→ 통계/조사 → 가계금융조사 관련 보고서 검색
→ 보고서 내용 확인
→ 기준이 없으면 "일반적인 금융 조언" 또는 "금융권 권장 기준"으로 변경
```

---

### 3. 비상금 6개월치 생활비 권장

**현재 문서 내용**:

- "금융감독원: 비상금 권장 기준 (6개월치 생활비)"
- "일반적인 금융 조언: 3-6개월치 생활비 준비 권장"

**검증 필요 사항**:

- [ ] 금융감독원에서 공식적으로 "6개월치 생활비"를 권장하는 문서 확인
- [ ] 금융감독원이 아닌 다른 기관(한국은행, 금융권 일반)의 권장사항인지 확인

**주의사항**:

- 6개월치 비상금은 일반적인 금융 조언일 수 있음
- 금융감독원 공식 권장사항인지 확인 필요

**권장 조치**:

```
금융감독원 공식 사이트에서 비상금 관련 권장사항 검색
→ 공식 문서가 없으면 "금융권 일반 권장 기준"으로 변경
→ 또는 "한국은행, 금융권 권장 기준"으로 변경
```

---

### 4. 통계청 생명표 기대여명 20년

**현재 문서 내용**:

- "통계청 생명표: 기대여명"
- "국민연금공단: 평균 기대여명 약 20년 (60세 기준)"

**검증 필요 사항**:

- [ ] 통계청 생명표에서 60세 기준 기대여명 확인
- [ ] 남성/여성 구분 여부 확인
- [ ] 최신 연도(2024 또는 2025) 데이터 확인

**권장 조치**:

```
통계청 KOSIS (https://kosis.kr/) 접속
→ 통계 → 인구 → 생명표
→ 60세 기준 기대여명 확인
→ 남성/여성 구분 또는 평균값 사용 결정
→ 문서 업데이트
```

---

### 5. 경제 위기 시나리오 (2008, 2020)

**현재 문서 내용**:

- "2008 금융위기: 주식 시장 약 50% 하락 (참고)"
- "2020 코로나19: 주식 시장 약 30% 하락 (참고)"

**검증 필요 사항**:

- [ ] 코스피 지수 2008년 최고점 대비 최저점 하락률 확인
- [ ] 코스피 지수 2020년 최고점 대비 최저점 하락률 확인
- [ ] 정확한 수치로 문서 업데이트

**권장 조치**:

```
한국거래소 또는 금융 데이터 제공 사이트 접속
→ 코스피 지수 2008년, 2020년 데이터 확인
→ 최고점 대비 최저점 하락률 계산
→ 문서에 정확한 수치 반영
```

---

### 6. 통계청 임금 상승률 3-4%

**현재 문서 내용**:

- "통계청 임금근로시간조사"
- "최근 5년 평균 임금 상승률 약 3-4%"

**검증 필요 사항**:

- [ ] 통계청 임금근로시간조사에서 최근 5년 임금 상승률 확인
- [ ] 정확한 평균값 계산
- [ ] 2020-2024 또는 2019-2023 기간 확인

**권장 조치**:

```
통계청 KOSIS 접속
→ 통계 → 노동 → 임금근로시간조사
→ 최근 5년 임금 상승률 확인
→ 평균 계산 후 문서 업데이트
```

## 권장 수정 사항

### 1. 출처 표현 방식 개선

**현재**:

```
금융감독원 가계금융조사: 건강한 가계 기준
```

**권장 (검증 후)**:

```
금융감독원 가계금융조사 보고서 (YYYY년) 참고
또는
금융권 일반 권장 기준 (저축률 30% 이상 권장)
```

### 2. 불확실한 출처 명시

**현재**:

```
금융감독원: 비상금 권장 기준 (6개월치 생활비)
```

**권장 (검증 후)**:

```
금융권 일반 권장 기준: 6개월치 생활비 준비 권장
(출처: 금융권 일반 조언, 금융감독원 공식 문서 미확인)
```

### 3. 참고 기간 명시

**현재**:

```
기본값: 2.5% (최근 3년 평균)
```

**권장**:

```
기본값: 2.5% (2021-2023년 평균, 한국은행 ECOS 기준)
※ 실제 계산 시 최신 데이터로 업데이트 필요
```

## 참고 사항

1. **공식 통계 기관의 데이터는 정기적으로 업데이트됨**

   - Phase 1 개발 시점에 최신 데이터로 재확인 필요
   - Phase 2에서는 API를 통해 실시간 데이터 사용 권장

2. **일반적인 금융 조언 vs 공식 기준**

   - 공식 기준이 없는 경우 "금융권 일반 권장 기준"으로 명시
   - 출처를 명확히 구분하여 표시

3. **과거 사례 참고**
   - 2008, 2020 위기 사례는 "참고 사례"로 명시
   - 정확한 수치로 업데이트 필요
