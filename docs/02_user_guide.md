## 추가 확인 필요 사항

### 1. 초기 사용자 튜토리얼

**문제 상황**:

- 첫 방문 사용자가 어떻게 시작해야 할지 모름
- 각 기능의 목적과 사용법을 이해하기 어려움

**제안**:

#### 옵션 A: 단계별 가이드 (추천)

- 첫 방문 시 자동으로 튜토리얼 모달 표시
- 3-4단계로 구성:
  1. 환영 메시지 + 프로젝트 소개
  2. 입력 데이터 설명 (어디서 찾을 수 있는지)
  3. 결과 해석 방법
  4. 다음 단계 안내

#### 옵션 B: 인터랙티브 하이라이트

- 첫 방문 시 각 섹션에 하이라이트 표시
- "이것을 클릭하세요" 형태의 가이드
- 건너뛰기 옵션 제공

#### 옵션 C: 비디오 튜토리얼

- YouTube 또는 내장 비디오 플레이어
- 2-3분 분량의 사용법 설명
- 텍스트 가이드와 병행

**권장**: 옵션 A (단계별 가이드) + 옵션 C (비디오 링크)

**구현 예시**:

```python
# Streamlit에서 튜토리얼 구현
if 'tutorial_completed' not in st.session_state:
    st.session_state.tutorial_completed = False

if not st.session_state.tutorial_completed:
    with st.expander("📚 처음 사용하시나요? 튜토리얼 보기", expanded=True):
        st.markdown("""
        ### 환영합니다!
        이 도구는 여러분의 재정 상황을 분석하고 미래를 시뮬레이션합니다.

        **다음 단계:**
        1. 왼쪽 사이드바에서 기본 정보 입력
        2. '계산하기' 버튼 클릭
        3. 결과 확인 및 인사이트 읽기

        [자세한 사용법 보기](링크)
        """)

        if st.button("튜토리얼 완료"):
            st.session_state.tutorial_completed = True
            st.rerun()
```

### 2. 예시 데이터 제공

**문제 상황**:

- 실제 데이터 입력이 부담스러움
- 어떤 값이 적절한지 모름
- 테스트해보고 싶지만 데이터가 없음

**제안**:

#### "샘플 데이터로 시도해보기" 기능

- 버튼 클릭 시 예시 데이터 자동 입력
- 여러 시나리오 제공:
  - **시나리오 1: 신입 직장인**
    - 나이: 25세, 연봉: 3,000만원, 자산: 500만원
  - **시나리오 2: 중년 직장인**
    - 나이: 40세, 연봉: 6,000만원, 자산: 5,000만원
  - **시나리오 3: 은퇴 준비 중**
    - 나이: 55세, 연봉: 8,000만원, 자산: 3억원

#### 구현 방법

```python
# 사이드바에 샘플 데이터 버튼
st.sidebar.markdown("---")
st.sidebar.markdown("### 🧪 샘플 데이터로 시도해보기")

sample_scenarios = {
    "신입 직장인": {
        'current_age': 25,
        'retirement_age': 60,
        'salary': 3000,
        'monthly_expense': 150,
        'total_assets': 500,
        'total_debt': 0
    },
    "중년 직장인": {
        'current_age': 40,
        'retirement_age': 60,
        'salary': 6000,
        'monthly_expense': 300,
        'total_assets': 5000,
        'total_debt': 2000
    },
    "은퇴 준비 중": {
        'current_age': 55,
        'retirement_age': 60,
        'salary': 8000,
        'monthly_expense': 400,
        'total_assets': 30000,
        'total_debt': 5000
    }
}

selected_scenario = st.sidebar.selectbox(
    "시나리오 선택",
    list(sample_scenarios.keys())
)

if st.sidebar.button("샘플 데이터 적용"):
    # 세션 상태에 샘플 데이터 저장
    for key, value in sample_scenarios[selected_scenario].items():
        st.session_state[key] = value
    st.rerun()
```

**사용자 경험**:

- "내 데이터는 아직 준비 안 됐는데..." → 샘플로 먼저 체험
- 실제 데이터 입력 전에 도구 이해
- 여러 시나리오로 다양한 결과 확인

### 3. 도움말 및 툴팁

**문제 상황**:

- 각 입력 필드의 의미가 불명확
- 어떤 값을 입력해야 할지 모름
- 전문 용어 이해 어려움

**제안**:

#### 입력 필드별 도움말

각 입력 필드 옆에 "?" 아이콘 또는 정보 아이콘 추가

| 입력 항목      | 도움말 내용                                       |
| -------------- | ------------------------------------------------- |
| 현재 나이      | 현재 만 나이를 입력하세요                         |
| 기대 은퇴 나이 | 은퇴를 계획하는 나이입니다                        |
| 연봉           | 세전 연봉 (단위: 만원). 보너스 포함 여부는 선택   |
| 월 지출        | 월 평균 지출액 (단위: 만원). 고정비 + 변동비 포함 |
| 현재 총 자산   | 예금, 적금, 주식, 부동산 등 모든 자산의 합계      |
| 현재 총 부채   | 대출, 신용카드 빚 등 모든 부채의 합계             |

#### 구현 방법

```python
# Streamlit에서 툴팁 구현
import streamlit as st

def help_tooltip(text):
    """도움말 툴팁"""
    st.markdown(f"""
    <div style="display: inline-block;">
        <span title="{text}">ℹ️</span>
    </div>
    """, unsafe_allow_html=True)

# 사용 예시
col1, col2 = st.columns([3, 1])
with col1:
    salary = st.number_input("연봉 (만원)", min_value=0)
with col2:
    help_tooltip("세전 연봉을 입력하세요. 보너스 포함 여부는 선택입니다.")
```

#### 상세 도움말 페이지

- 별도 페이지 또는 확장 가능한 섹션
- FAQ 형식
- 스크린샷 포함

### 4. 결과 해석 가이드

**문제 상황**:

- 계산 결과를 봐도 무슨 의미인지 모름
- 다음에 무엇을 해야 할지 모름

**제안**:

#### 결과 페이지에 해석 가이드 포함

- 각 인사이트 아래에 "이것이 의미하는 것은?" 섹션
- 실행 가능한 액션 아이템 제시

**예시**:

```
📊 결과: 현재 소비 패턴을 유지하면 10년 후 약 5,000만원이 남습니다.

💡 이것이 의미하는 것은?
- 현재 소비 수준을 유지해도 자산이 증가합니다
- 하지만 목표 자산에 도달하기에는 부족할 수 있습니다

✅ 다음 단계:
1. 지출을 10% 줄이면 10년 후 7,000만원이 됩니다
2. 연봉 인상률을 5%로 가정하면 10년 후 8,000만원이 됩니다
3. 투자를 시작하면 더 빠르게 자산을 늘릴 수 있습니다
```

### 5. 단계별 진행 표시

**문제 상황**:

- 입력이 많아서 어디까지 했는지 모름
- 다음에 무엇을 입력해야 할지 모름

**제안**:

#### 진행 상황 표시

- 상단에 진행 바 표시
- 단계별 체크리스트

**구현 예시**:

```python
# 진행 상황 표시
st.progress(0.6)  # 60% 완료
st.markdown("""
**입력 진행 상황:**
- ✅ 기본 정보 (나이, 은퇴 나이)
- ✅ 소득 정보
- ⏳ 소비 정보 (진행 중)
- ⬜ 자산 정보
- ⬜ 부채 정보
""")
```

## 구현 우선순위

### Phase 1 (필수)

1. ✅ 기본 도움말 (각 입력 필드 툴팁)
2. ✅ 샘플 데이터 기능
3. ✅ 간단한 환영 메시지

### Phase 2 (권장)

4. ⚠️ 단계별 튜토리얼
5. ⚠️ 결과 해석 가이드
6. ⚠️ 진행 상황 표시
7. ⚠️ 상세 FAQ 페이지

---

## 사용자 경험 원칙

1. **점진적 공개**: 처음에는 간단하게, 필요시 상세 정보 제공
2. **선택적 학습**: 튜토리얼은 건너뛰기 가능
3. **즉시 체험**: 샘플 데이터로 바로 사용 가능
4. **명확한 안내**: 다음 단계가 항상 명확
