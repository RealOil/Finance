- 연봉/자산/채무/연봉 인상률 등을 입력하면,
    
    → 인플레이션/환율 변화/투자 대비 수익률 등을 자동 계산
    
- “3년 전 비트코인을 샀다면 지금 얼마일까?”, “CMA에 넣었으면?” 등
    
    → 실시간 비교 시뮬레이션 제공
    
- 자산이 인플레이션 대비 얼마나 보존되었는지 시각화
- 다양한 경제적 상황(이사, 결혼, 실직 등)에서 돈의 역할을 설명해주는 구조


## 과제 구현 예시

| 카테고리 | 입력데이터 |
| --- | --- |
| 개인 정보 | 현재 나이, 기대 은퇴 나이 |
| 소득 | 연봉, 보너스, 연평균 소득 증가율 |
| 소비 | 월지출, 연간 고정지출, 라이프스타일 지출(여행/식비/쇼핑) |
| 금융 | 예금, 적금, 보험, 펀드, ETF, 주식, 암호화폐 |
| 부동산 | 매입가, 현재가치, 대출, 금리 |
| 외부 변수 | 환율, 인플레이션, 국가금리, 특정 자산군 지수 (코스피·나스닥 등) |

### **1) 인플레이션 기반 가치 비교 인사이트**

1. 지금 자산 총액의 **실질가치(물가 반영 후 가치)**
2. 최근 10년 평균 인플레이션 대비 **내 자산 성장률**
3. ‘만약 인플레이션만 반영했다면’ vs ‘내 실제 성장률’ 비교 그래프
4. 앞으로 10년 뒤 인플레이션 적용 시 **현재 자산의 예상 구매력**
5. 과거 5년 전 오늘 기준 **내가 같은 돈을 가지고 있었다면 구매력 차이**
6. 월급 인상률 vs 물가상승률 비교 (실질 월급 상승/감소 여부)

---

### **2) 소득 및 지출 기반 인사이트**

1. 현재 소비 습관 처리 시 **10년 뒤 남는 자산 추정치**
2. 소득 대비 지출 비율과 **재정 건전성 등급 평가**
3. 소비 패턴 변경 시나리오별 결과 (예: 지출 10% 줄이면 5년 뒤 얼마+)
4. 소득 증가율이 물가상승률보다 낮으면 위험 경고
5. 연봉 5%, 10%, 제자리 기준 **미래 자산 비교 시뮬레이션**

---

### **3) 투자 시나리오 기반 인사이트**

1. 5년 전 특정 ETF/주식을 샀다면 지금 가치
2. 동일 기간 비트코인/이더리움 샀을 경우 비교
3. 달러 환율 기준 외화투자 ROI
4. 코스피 vs 나스닥 vs 비트코인 vs 금 비교 성과
5. 투자하지 않고 예금만 했다면 결과
6. **리스크 대비 수익률: 샤프 지수 기반 내 투자 효율 등급**

---

### **4) 부동산 기반 분석**

1. 과거 대비 현재 매입 부동산 실질가치 변화
2. 대출금리 변동에 따른 손익 비교
3. 월세 전환 시 break-even point
4. 10년 후 예상 자산가치(국가별/도시별 평균 상승률 적용)
5. 매도/보유 전략 비교 시나리오

---

### **5) 환율·글로벌 변수 인사이트**

1. 원화가치 하락 대비 자산 잠식률
2. 동일 시점 미국/일본/유로존 구매력 비교
3. 해외 ETF 투자 대비 환차손/환차익 자동 계산
4. 특정 시점 환율로 환전했을 때의 대체 시나리오(“그때 샀다면…”)

---

### **6) 보험 및 장기 금융상품 인사이트**

1. 내가 지금까지 낸 보험료 총액 vs 보장 가치 비교
2. 동일 금액 투자 대비 보험 효율 평가
3. 중도 해지 vs 유지 시 재정 손익
4. 보험료가 20년 뒤 인플레이션 반영 시 얼마나 비싸지는가

---

### **7) 미래 리스크 및 시나리오 경고**

1. 은퇴 시점 월 지출 유지 가능 여부
2. 경제 위기(예: -30% 하락 시나리오) 적용 시 생존력
3. 소득 중단 상황(6개월·1년) 버틸 수 있는 기간
4. 주택담보대출 금리 2% 상승 시 부담 변화
5. 장기 의료비 증가 추정 대비 자산 위험도

---

### **8) 기회비용 기반 분석**

1. **“만약 그때 안 썼다면?”** 소비 → 투자 전환 시 결과
2. 흡연, 외식, 쇼핑 등 특정 소비 습관 비용의 10년 누적 영향
3. 월 20만 원씩 ETF vs 예금 비교 10년 시뮬레이션
4. 여행·명품 소비 vs 투자했을 경우 미래격차 그래프

---

### **9) 경제적 체력 지표**

1. 자산 대비 부채 비율
2. 순자산 성장 속도 → “성장 체력 점수”
3. 목표 자산 규모까지 남은 시간(페이스 분석)
4. 지출-투자 균형 비율 최적 모델 제안
5. 금융 스트레스 점수 (부채/고정비 기반)

---

### **10) 목표 설계 및 전략 추천**

1. 현재 데이터 기반 1년/3년/5년/10년 목표 시뮬레이션
2. 은퇴 시 월 생활비 유지하려면 필요한 목표 자산
3. 나와 비슷한 연령 자산 비교(백분위 분석)
4. 지금 내 자산 포트폴리오의 예상 CAGR(연평균 복리 성장률)
5. 위험 성향 기반 맞춤 자산 분배 추천
6. “지금 바로 해야 할 5가지 금융 행동 리스트”
